# Design Smells

- **Rigidity** — one change forces cascading changes.
- **Fragility** — changes break unrelated things.
- **Immobility** — components can't be reused due to tangling.
- **Needless complexity** — YAGNI violations, speculative architecture.
- **Needless repetition** — DRY violations.
- **Opacity** — code hard to understand.
